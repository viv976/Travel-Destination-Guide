Beyond Miles — Travel Destination Guide
Overview
Beyond Miles is a simple and elegant travel destination guide built using HTML and CSS.  
It provides users with an engaging interface to explore various travel destinations, learn about their unique features, and get inspired for their next trip.  
This project focuses on clean design, readability, and responsiveness — all without using JavaScript.

Features
1. Home section with a get in touch button
2. Gallery
3. About the company
4. featured destinations
5. sign up page

Tech Stack
- HTML5 — for semantic and structured content  
- CSS3 — for styling, layout, and responsive design
